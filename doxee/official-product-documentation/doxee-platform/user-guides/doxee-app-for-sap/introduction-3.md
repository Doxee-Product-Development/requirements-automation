Doxee App for SAP enables users to create documents directly in the CRM. The business user can create, edit, send or print documents personalized with CRM data with few clicks. The Doxee app for SAP benefits from the whole Doxee technology platform enabling to adapt the user workflow based on their needs. With the Mashup feature the app can be added to any object entity in the Sales or Service Cloud. For the power user the Doxee App for SAP offers an intuitive and powerful template designer embedded directly in the CRM system. With the help of Query Builder, which is an integral part of the app, the user can map the CRM data with the dynamic fields in the template in a quick, user-friendly way. In this manual we will show you how to configure and use the app.
