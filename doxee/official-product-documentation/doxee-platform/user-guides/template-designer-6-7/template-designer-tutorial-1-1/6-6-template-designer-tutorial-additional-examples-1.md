Navigate to other release versions of Doxee Platform Template Designer

[6.6](https://docs.doxee.com/docs/en/template-designer-6-6)

[6.7](https://docs.doxee.com/docs/en/template-designer-6-7)

Additional examples can be found in *&lt;Infinica Template Designer directory&gt;/Examples/en/Intermediate*. *Billsummary *and *AccountBalance *are very similar to the previously described *Invoice Template*. The folder *Offer of Renewal *contains two examples for more complex PDF form elements, which also use JavaScript. The folder *Advanced *includes two examples (*EnergyLetter *+ *SLACalculation*), for templates which are optimised for the use with *Doxee Composer*. More information about PDF form elements is included in the user guide.
