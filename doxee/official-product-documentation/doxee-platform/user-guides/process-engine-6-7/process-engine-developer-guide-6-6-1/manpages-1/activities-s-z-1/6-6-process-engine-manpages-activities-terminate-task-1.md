(terminateTask, Module: [Tasks](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-tasks-1))

Calls the terminateTask operation. The task is canceled and set to the state terminated.

**Input Parameter:**

- taskId (text/plain)

ID of the task to terminate.

**Output Parameter:**

- task (text/xml)

The terminated task in XML format.
