(urlDisconnector, Module: [Base](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-base-1))

Closes a connection to a filesystem URL. If no connection to the URL is open, a resource_access.io.connection.closed exception is thrown.

**Input Parameter:**

- url (x-infinica/url)

The URL of the server to which the connection will be closed.
