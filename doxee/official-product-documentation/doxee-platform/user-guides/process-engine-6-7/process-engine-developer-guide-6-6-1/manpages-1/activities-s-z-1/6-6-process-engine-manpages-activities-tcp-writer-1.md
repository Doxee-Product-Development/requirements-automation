(tcpWriter, Module: [Base](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-base-1))

Writes input data to TCP/IP socket.

**Input Parameters:**

- input (x-infinica/stream) (1-unbounded)

The input data to be written to the target socket.

- address (text/plain)

Target IP or host name.

- port (x-xsd/integer) Target port.
