(base, url)

([http://www.infinica.com/](http://www.infinica.com/), Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core-1))

Relativises a URL using a specified base URL. If the URL is a sub URL of the base URL, the relative path from the base URL to the sub URL will be returned, otherwise the unmodified URL will be returned.

**Parameters:**

- **base**

Base URL.

Type:

x-infinica/url

- **url**

URL to relativise.

Type:

x-infinica/url
