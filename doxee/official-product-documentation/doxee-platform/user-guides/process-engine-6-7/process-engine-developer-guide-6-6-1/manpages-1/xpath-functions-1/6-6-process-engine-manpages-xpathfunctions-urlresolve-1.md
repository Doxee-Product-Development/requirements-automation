(base, url)

([http://www.infinica.com/](http://www.infinica.com/), Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core-1))

Resolves a relative URL using the specified base URL. If the URL to resolve is already absolute, it will not be changed. 

**Parameters:**

- **base**

Base URL. Number of Base URLs must be 1 or equal to the number of URLs to resolve.

Type:

x-infinica/url

- **url**

URL to resolve.

Type:

x-infinica/url
