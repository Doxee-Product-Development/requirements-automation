(xml, xpathEntry, xpathKey, xpathValue)

([http://www.infinica.com/](http://www.infinica.com/), Module: [Base](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-base-1)) Transforms an xml into a properties string.

**Parameters:**

- **xml**

The xml to transform.

Type:

x-infinica/xml-node

- **xpathEntry**

An xpath query identifying the xml entries to use.

Type:

text/plain

- **xpathKey**

An xpath query identifying the property keys for the entries in the xml.

Type:

text/plain

- **xpathValue**

An xpath query identifying the Property values for the entries in the xml.

Type:

text/plain
