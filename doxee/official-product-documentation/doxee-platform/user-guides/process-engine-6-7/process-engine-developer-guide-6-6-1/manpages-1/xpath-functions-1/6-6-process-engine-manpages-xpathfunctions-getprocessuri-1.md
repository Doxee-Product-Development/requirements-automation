(absolute)

([http://www.infinica.com/](http://www.infinica.com/), Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core-1)) Returns the URL of the running process.

Deprecated: Use getProcessUrl instead.

**Parameter:**

- **absolute** (**0-1)**

Whether an absolute path should be returned. Defaults to false. A relative process path represents the relative URL that is used to instantiate the process. Usually this is the relative path from the process directory to the process definition file.

Type:

x-xsd/boolean
