([http://www.infinica.com/](http://www.infinica.com/), Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core-1)) Converts a value to a different type.

Note: The XPath processor internally, auto-converts some standard types losing Doxee's type information. For example, URLs are automatically converted to strings. For situations where the original Infinica type must be preserved, the function convertVariable can be used instead.

**Parameters:**

- **value**

Source value.

- **type**

Type name for the target value.
