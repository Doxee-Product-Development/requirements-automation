(deleteTasks, Module: [Tasks](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-tasks-1))

Calls the deleteTasks operation. Deletes all tasks which are found by the provided task query.

**Input Parameter:**

- query (text/xml)

The task query used to find the tasks which should be deleted.

**Output Parameter:**

- taskIds (text/plain) (0-unbounded)

A list of IDs for all the tasks which have been deleted.
