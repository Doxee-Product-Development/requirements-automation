(findTasks, Module: [Tasks](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-tasks))

Calls the findTasks operation. Executes the provided query and retrieves all tasks it returns.

**Input Parameter:**

- query (text/xml)

The task query which should be executed by the task manager.

**Output Parameter:**

- tasks (text/xml)

list of tasks found by the query, returned in XML format.
