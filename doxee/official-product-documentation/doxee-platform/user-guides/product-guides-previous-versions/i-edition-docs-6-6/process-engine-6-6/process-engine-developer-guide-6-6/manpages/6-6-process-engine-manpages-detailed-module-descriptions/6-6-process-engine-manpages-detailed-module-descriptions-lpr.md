Prints documents via the LPR protocol.

Activity:	

[LPR Printer](/doxee-platform/docs/lpr-printer)

The lprPrinter activity sends a document to a printer via the LPR (Line Printer Daemon/Line Printer

Remote) protocol. The activity's input parameters correspond with the parameters supported by the protocol.

For more information regarding the LPR protocol, please consult the official specification in RFC 1179.
