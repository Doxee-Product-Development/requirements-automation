(Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core)) Aliases: x-infinica/uri

A URL (Uniform Resource Locator), colloquially termed a web address, points to any (web) resource, which the Process Engine typically can access to read and write data. Most commonly, URLs start with http(s):// or with file:///.

**Conversions:**

text/plain → x-infinica/url 

x-infinica/url → text/plain 

x-infinica/file → x-infinica/url 

x-infinica/url → x-infinica/file

x-infinica/stream → text/plain → x-infinica/url
