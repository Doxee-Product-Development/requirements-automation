Navigate to other release versions of Doxee Platform Process Engine

[6.6](https://docs.doxee.com/docs/en/process-engine-6-6)

[6.7](https://docs.doxee.com/docs/en/process-engine-6-7)

(urlRevert, Module: [Resource Access](/doxee-platform/docs/resource-access)) 

Reverts one or more versioned resources.

**Input Parameters:**

- url (x-infinica/url) (1-unbounded) 

Resource URLs.

- targetUrl (x-infinica/url) (1-unbounded)

URLs of the versions to revert to. The number of values specified here must be the same as the number of values for the url parameter.
