Navigate to other release versions of Doxee Platform Process Engine

[6.6](https://docs.doxee.com/docs/en/process-engine-6-6)

[6.7](https://docs.doxee.com/docs/en/process-engine-6-7)

 [Base](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-base) 

base, 6.6.2.0

 Common activities and XPath functions required for many processes.

[Command Line Main Module](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-command-line-module) 

cmd, 6.6.2.0

The command line version of the Doxee Process Engine. Can be used to run a single process or can be started as a standalone server.

[Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core) 

core, 6.6.2.0

Core functionality required by every Process Engine installation.

[Database Logging](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-database-logging) 

database-logging, 6.6.2.0

Provides a logger for writing logging output to an SQL database.

[E-Mail](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-e-mail) 

email, 6.6.2.0

Provides an activity for sending emails via an SMTP server.

[FO Concatenation](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-fo-concatenation) 

fo-concatenation, 6.6.2.0

[FOP](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-fop) 

fop, 6.6.2.0

Support for the Apache FOP renderer. FOP is the standard way to render PDF documents in Doxee.

[JPA Storage](/doxee-platform/docs/jpa-storage) 

storage-jpa, 6.6.2.0

[JSON Support](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-json-support) 

json, 6.6.2.0

Provides a JSON data type with conversions to text and XML types.

[Log4j Support](/doxee-platform/docs/log4j-support) 

log4j, 6.6.2.0

Logging via Log4j 2.

[Log Analysis](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-log-analysis) 

log-analysis, 6.6.2.0

Provides activities for analysing Doxee log-files. All activities can read log entries from files and strings and create an XML containing the results.

[LPR](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-lpr) 

lpr, 6.6.2.0

Prints documents via the LPR protocol.

[Messaging](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-messaging) 

messaging, 6.6.2.0

Provides functionality for communicating via JMS messages. Includes a logger for writing logging output as messages, a message activity for sending text messages and a wait state which waits for a certain message.

[Mongo Storage](/doxee-platform/docs/mongo-storage) 

storage-mongo, 6.6.2.0

[OpenPDF](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-openpdf)

(openpdf, 6.6.2.0)

Provides activities for using and modifying PDFs via OpenPDF (a fork of iText).

[PDF/A Validation](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-pdfa-validation) 

pdfa-validation, 6.6.2.0

Validates PDF/A documents against various PDF/A standards using the veraPDF library.

[PDFBox](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-pdfbox) 

pdfbox, 6.6.2.0

PDF utility activities based on the PDFBox library.

[PDF Forms](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-pdf-forms) 

pdf-forms, 6.6.2.0

[PDF Renderer](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-pdf-renderer) 

pdf-renderer, 6.6.2.0

Renders and prints PDF documents. This module uses Sun's obsolete PdfRenderer library. It is recommended to use the pdfbox module instead, which provides the same functionality.

[PostScript Handler](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-postscript-handler) 

postscript-handler, 6.6.2.0

[Process Engine Reporting](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-process-engine-reporting) 

reporting, 6.6.2.0

This module provides activities which gather data for the Infinica reporting functionality.

[Process Engine REST Service Client](/doxee-platform/docs/process-engine-rest-service-client) 

process-engine-rest-client, 6.6.2.0

[Process Engine SOAP Service Client](/doxee-platform/docs/process-engine-soap-service-client) 

process-engine-soap-client, 6.6.2.0

[Processes](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-processes) 

processes, 6.6.2.0

Provides activities for accessing the Process Engine API.

[Redis Storage](/doxee-platform/docs/redis-storage) 

storage-redis, 6.6.2.0

[Resource Access](/doxee-platform/docs/resource-access) 

resource-access, 6.6.2.0

Provides access to files and directories in the local file system, Infinica Content Repository, or remote servers.

[REST](/doxee-platform/docs/rest) 

rest, 6.6.2.0

Support for calling REST services.

[SAP](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-sap) 

sap, 6.6.2.0

SAP support.

[Service](/doxee-platform/docs/service) 

service, 6.6.2.0

Service core module. Runs the Process Engine as a system service.

[SFTP](/doxee-platform/docs/service-1) 

sftp, 6.6.2.0

Support for SFTP connections.

[SOAP](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-soap) 

soap, 6.6.2.0

Support for calling SOAP services.

[Splitter](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-splitter) 

splitter, 6.6.2.0

[Tasks](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-tasks) 

tasks, 6.6.2.0

This module contains task activities, each of which calls a particular operation of the Task Manager API. All calls are executed on the task manager configured for the Process Engine, which executes the current process. The result is the same as when calling the API directly via SOAP or REST.

[Template Conversion](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-template-conversion) 

template-conversion, 6.6.2.0

[User Access](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-user-access) 

users, 6.6.2.0

Support for accessing user data.

[Zip4j](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-zip4j)
