**Parameters:**

- **xml**

The xml to transform.

Type:

x-infinica/xml-node

- **xpathEntry**

An xpath query identifying the xml entries to use.

Type:

text/plain

- **xpathKey**

An xpath query identifying the property keys for the entries in the xml.

Type:

text/plain

- **xpathValue**

An xpath query identifying the Property values for the entries in the xml.

text/plain
