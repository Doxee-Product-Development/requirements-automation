(urlConnector, Module: [Base](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-base)) Initializes a connection to a filesystem URL.

**Input Parameters:**

- url (x-infinica/url)

The URL of the server to which a connection will be established.

- password (x-infinica/secret-text) (0-1)

Optional password for the connection, if not encoded in the userinfo part of the URL.
