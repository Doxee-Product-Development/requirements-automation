(url)

([http://www.infinica.com/](http://www.infinica.com/), Module: [Core](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-core))

Returns the cache user key for the specified target URL. The user key is based on the credentials that would be used to access the URL, depending on the credentials registry and, possible, the current user.

**Parameter:**

- **url**

Target URL.

Type:

x-infinica/url
