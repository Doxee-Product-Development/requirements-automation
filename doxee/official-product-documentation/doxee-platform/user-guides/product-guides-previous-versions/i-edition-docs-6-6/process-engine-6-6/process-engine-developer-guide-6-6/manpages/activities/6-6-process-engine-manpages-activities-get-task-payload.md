(getTaskPayload, Module: [Tasks](/doxee-platform/docs/6-6-process-engine-manpages-detailed-module-descriptions-tasks))

Calls the getPayload operation. Retrieves the task payload for the task with the provided ID.

**Input Parameter:**

- taskId (text/plain)

ID of the task for which the payload should be retrieved.

**Output Parameter:**

- All variables which are returned with the payload are available as output parameters.
