Each Project has a universal unique system identifier or **UUID**, which takes the form of an immutable alphanumeric string, for example, `90052299-a1c4-4d79-ae00-830b5cc05597`. The UUID may be used in various configurations, for example, when embedding a Project in a workflow.

It can be retrieved in one of the following ways:

- From the **Project settings**

- From the **Storyboard editor**

**From the Project settings:**

- Find the Project you want to adjust and open its **Project settings**.

- Go to **General** &gt; **UUID**. The UUID of this Project can be viewed here.

- Click the **copy icon** next to the UUID to automatically copy it to your clipboard.

**From the Storyboard editor:**

- Find the Project you want to adjust and open its **Storyboard editor**.

- In the top toolbar, click the **copy button** next to the Project name to automatically copy the UUID to your clipboard.
