You can choose up to two different fonts (primary and secondary) for every Project. These will be available as font options in the video editor for all text elements.

- **Primary font**: Default font of new text elements

- **Secondary font**: Second font option for text elements

To select a new primary or secondary font:

- Open the **Project settings** &gt; **Layout &amp; Sound** and scroll to the section **Fonts**.

- Select one of the options from the drop-down menus and click anywhere to confirm.

> Custom fonts

Itâ€™s possible to upload a custom font and add it as an option for primary or secondary fonts. To do this, open the **Project settings** &gt; **Layout &amp; Sound **and navigate to the section **Fonts**. Click the **Upload new** button and select a font file (*.ttf or *.otf) from your device. The custom font will be available as an option in the drop-down menus for primary and secondary fonts.
