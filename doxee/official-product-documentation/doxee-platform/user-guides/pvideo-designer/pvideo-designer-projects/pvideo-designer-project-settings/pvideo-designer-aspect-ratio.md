This setting changes the aspect ratio of the video frame.

To change the aspect ratio:

- Open the **Project settings** &gt; **Layout &amp; Sound** and scroll to the section **Aspect ratio**.

- Select an option from the drop-down menu and click anywhere to confirm.

> Warning

Changing the aspect ratio of a Project which already contains content can lead to placement issues with elements. Itâ€™s best practice to adjust this setting as a first step and before creating any video content.
