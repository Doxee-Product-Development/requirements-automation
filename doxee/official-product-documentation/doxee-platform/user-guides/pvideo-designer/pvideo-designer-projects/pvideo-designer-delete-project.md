Deleting an existing Project can be done in one of the following ways:

- **From the Projects tab**

- **From the Project editor**

**From the Projects tab:**

- Navigate to the **Projects tab** and find the Project you want to duplicate.

- Open the **three-dot menu** and click **Delete**.

**From the Project editor:**

- Open the **Project editor**.

- Open the **hamburger menu** in the **toolbar** and click **Delete**.

The current Project, including all scenes and pop-ups in the sidebar and all Storyboards, will be removed from Video Designer.

> Irreversible action
