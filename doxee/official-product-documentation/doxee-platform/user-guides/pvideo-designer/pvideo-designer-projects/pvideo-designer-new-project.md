To create a new Project in Video Designer, follow these steps:

- **Access the Projects tab:** Open the Video Designer application and go to the **Projects** tab in the left sidebar for an overview of existing projects.

- **Create a new Project:** Click the **New Project** button to create a new entry and open it in the **Storyboard editor**.

- **Add scenes and put together a Storyboard:** Add and edit scenes in the **Scene editor** and connect them in the **Storyboard editor** to create a cohesive video.

- **Save the Project:** Regularly save your project using the **Save** option in the top toolbar.

- **Configure the mapping:** Click the **Data Mapper** button and configure mappings to link video elements to data sources for interactivity.

- **Produce an interactive video:** Click the **Production** button to create and preview the finished Pvideo.

> Tip: Getting started with Video Designer

The entire editing process can be seen in action in [Getting started: Create a Pvideo with Video Designer](/doxee-platform/docs/pvideo-designer-getting-started).
