Duplicating an existing Project can be done in one of the following ways:

- **From the Projects tab**

- **From the Project editor**

**From the Projects tab:**

- Navigate to the **Projects tab** and find the Project you want to duplicate.

- Open the **three-dot menu** and click **Duplicate**.

**From the Project editor:**

- Open the **Project editor**.

- Open the **hamburger menu** in the **toolbar** and click **Duplicate**.

A copy of the Project including all scenes and pop-ups in the sidebar and all Storyboard will be created.
