Mappings can be created in one of the following ways:

- From the **Projects tab**

- From the **Storyboard editor**

**From the Projects tab:**

- Navigate to the **Projects** tab.

- Find the Project you want to create and hover over its table row.

- Click the **three-dot button** that appears on the right side.

- In the three-dot menu, click **Data Mapper** &gt; **Create new schema**.

**From the Storyboard editor:**

- Open a Project in the **Storyboard editor**.

- In the top toolbar, click **Data Mapper** &gt; **Create new schema**.

A new schema is created and the Mapping editor is opened.
