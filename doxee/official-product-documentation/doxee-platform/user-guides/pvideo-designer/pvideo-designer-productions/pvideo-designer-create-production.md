A new Production of an interactive video can be started in one of the following ways:

- **From the Projects tab**

- **From the Project editor**

**From the Projects tab:**

- Navigate to the **Projects tab** and find the Project you want to duplicate.

- Open the **three-dot menu** &gt; **Production**.

**From the Project editor:**

- Open the **Project editor**.

- In the **toolbar**, click the action button **Production**.

The Production pipeline is opened and the next steps in the process can be configured.
