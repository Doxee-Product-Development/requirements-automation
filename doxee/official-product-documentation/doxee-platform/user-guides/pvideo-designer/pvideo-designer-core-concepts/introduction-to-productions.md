A **Production** of an interactive video is the runtime resolution of the preconfigured video template together with any possible data mappings. The result is the finalized Pvideo, which can be delivered to the recipient through a personalized URL (Purl) that is generated automatically and simultaneously.

### FAQ
#### What is the difference between scenes and pop-ups?
Scenes and pop-ups are the basic building blocks of interactive videos and can contain the same elements. However, pop-ups can only be integrated into the timeline of an interactive video by mapping them to CTA buttons.
