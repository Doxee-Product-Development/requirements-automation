## **Description**
The **Length **operator allows to determine the length of an input string.

## **Input**
**Input**

**Data type**

**Description**

**in0**

String

The string whose length you want to calculate

## ***Output***
**Output**

**Data type**

**Description**

**out0**

Number

The length (i.e. number of characters) of the input string

## **Settings**
None.

## **Special cases**
In special cases, the operator behaves as follows:

**Case**

**Behavior**

**NULL input (no input)**

The operator returns the value ***-1***
