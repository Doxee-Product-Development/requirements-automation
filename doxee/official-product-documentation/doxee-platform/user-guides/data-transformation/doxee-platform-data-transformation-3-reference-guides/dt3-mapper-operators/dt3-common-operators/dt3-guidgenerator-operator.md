## **Description**
The **GUIDGenerator **operator allows to generate a unique GUID number when receiving any type of input data.

## **Input**
**Input**

**Data type**

**Description**

**in0**

Any

The input data that you want to use to trigger the generation of a unique GUID number

## **Output**
**Output**

**Data type**

**Description**

**out0**

Number

Unique GUID number

## **Settings**
None.

## **Special cases**
In special cases, the operator behaves as follows:

**Case**

**Behavior**

**NULL input (no input)**

The operator does not return any value in output.
