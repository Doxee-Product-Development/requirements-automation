Navigate to other release versions of Doxee Platform Business Designer

[6.6](https://docs.doxee.com/docs/en/business-designer-6-6)

[6.7](https://docs.doxee.com/docs/en/business-designer-6-7)

The following shortcuts are available:

**Keys**

**Description**

Ctrl + S

Save opened file

Shift + Ctrl + S

Save file as

Alt + Ctrl + S

Save all opened files

Ctrl + O

Open New File

Ctrl + C

Copy selection

Ctrl + X

Cut selection

Ctrl + V

Paste content from clipboard (due to technical limitations this shortcut is recommended by Doxee)

Ctrl + Z

Undo last action

Ctrl + Y

Redo last action

Ctrl + A

Select all (editor)

Alt + Up

Move up (same level in Outline)

Alt + Down

Move down (same level in Outline)

Ctrl + Shift + Up

Move up as child

Ctrl + Shift + Down

Move down as child

Alt + Shift + Up

Move up as parent

Alt + Shift + Down

Move down as parent

Ctrl + K

Close opened tab

Alt + Ctrl + K

Close all opened tabs

Home

Move the Caret from current position to the start of the row

Shift + Home

Select the text from current position to the start of the row

Ctrl + Home

Move the Caret from current position to the start of the template

Shift + Ctrl + Home

Same as "Ctrl + Home" but will also create Selection

End

Move the Caret from current position to the end of the row

Shift + End

Select the text from current position to the end of the row

Ctrl + End

Move the Caret from current position to the End of the template

Shift + Ctrl + End

Same as "Ctrl + End" but will also create Selection

Ctrl + Right

Move the Caret from current position to the start of the next word

Shift + Ctrl + Right

Select the text from current position to the start of the next word

Ctrl + Left

Move the Caret from current position to the start of the current or previous word

**Keys**

**Description**

Shift + Ctrl + Left

Select the text from current position to the start of the current or previous word

Page Up

Move the Caret from current position one "page" up

Shift + Page Up

Select the text from current position one "page" up

Page Down

Move the Caret from current position one "page" down

Shift + Page Down

Select the text from current position one "page" down

Alt + 0

Accessibility instructions modal window

Alt + Shift + S

Styles

Alt + Shift + E

Palette

Alt + Shift + H

Help

Alt + Shift + D

Data Input

Alt + Shift + O

Output Configuration

Alt + Shift + R

Search &amp; Replace

> Note

Use cmd instead of Ctrl when using an Apple keyboard.
