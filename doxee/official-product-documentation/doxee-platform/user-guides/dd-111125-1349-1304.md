## Supported browsers
### Doxee Platform 2
> **Supported browsers**

For the best user experience, we recommend using the latest stable version of one of the following browsers to access Doxee Platform services:

- **Google Chrome**

- **Microsoft Edge Chromium **

- **Mozilla Firefox**

The following browsers are not officially supported:

- **Microsoft Edge (non-chromium) **(EOL 9 March 2021) 

- **Microsoft Internet Explorer 11 **(EOL 17 August 2021) 

- **Safari**

### Doxee Platform 3
> **Supported browsers**

For the best user experience, we recommend using the latest stable version of one of the following browsers to access Doxee Platform portals:

- **Windows 10 (Desktop): Google Chrome **

- **macOS 10 (Desktop): Safari**

The recommended resolutions for viewing portals are:

- **1680x1050**

- **1920x1080**

> **EOL Flash Player**
