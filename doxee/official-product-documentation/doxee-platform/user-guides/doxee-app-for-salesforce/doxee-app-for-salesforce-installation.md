## Prerequisites
The following software and resources are required for installing the Doxee App for Salesforce®:

- **Salesforce**

- **Doxee i-Edition**

## Installation
> How to get the installation link for the Doxee App for Salesforce®

Contact your Doxee project consultant to receive the installation link for the licensed version of Doxee App for Salesforce®.

- Open the installation link in the browser.

- Enter your administrator username and password for the Salesforce organization in which you want to install the package and then click **Log In**.

- Select the first box: choose an audience for this installation.

- Check the **Acknowledgment** checkbox provided above the “Install” button. 

- Click **Install** and wait for the installation to be completed. You’ll see a message describing the progress and a confirmation message at the completion of installation. If the installation takes longer than one minute, Salesforce will send you an email once completed.
