### FAQ
#### Where can I find the documentation of a previous release version?
Some components are available as on-premise installations. The full documentation for each release version can be found under the category **Product guides** &gt; **Previous versions**.

A list of currently supported release versions can be found under [supported software versions](/doxee-platform/docs/release-history-and-supported-versions).

#### Where can I find the documentation of the legacy Doxee Platform® 2?
Some components of Doxee Platform&reg; 2, like **Tracking &amp; Reporting** and **Staging &amp; Aggregation**, may be integrated with Doxee Platform&reg; 3 for backwards compatibility. The documentation of these components can be found on Doxee Platform® Documentation.

Workplace is a web-based application for executing interactive document workflows, so-called Business Cases. It provides an interface for opening a document template, editing it, and delivering the finished document through various channels.

Composer is a web-based application for editing document templates which were created with Business Designer. It is used mainly to resolve dynamic fields into real data read from a database.

Process Engine is a framework for defining and executing processes. These perform operations by executing activities, which are predefined program elements designed for specific tasks like reading and writing files, rendering templates or sending communications. Users can design processes by arranging activities and combining them with input data. Process Engine can be called to execute a process through a variety of interfaces, for example command line, API call, and embedded Java interfaces.

Process Designer is a GUI tool for creating and testing Infinica Process Definitions (IPD). It&#39;s a support tool for Process Engine, providing a graphical editing and log interface and removing the necessity to manually write XML processes in a text editor.

Video Designer is a web-based video editing tool for creating interactive videos. It boasts a variety of innovative AI-powered features like script writing and text-to-speech (TTS). Video Designer is Doxee’s native solution for creating interactive videos and can be integrated seamlessly into the larger Doxee Platform® workflow ecosystem to cover the entire lifecycle of interactive videos from creation to delivery and archiving.

Business Designer is a web-based application for creating and managing templates for documents and communications. It supports static and dynamic content, layouts, and logic.

Doxee Platform&reg; is a cloud-native CCM and CXM platform offered by Doxee to manage, create, distribute, and analyze the communications produced by enterprise companies to their end customers through physical and digital channels. The Doxee Platform&reg;&#39;s features are made available through a set of software components and graphical interfaces available to the user.
